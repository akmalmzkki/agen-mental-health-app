## Tutorial Menjalankan Aplikasi Ini

1. Membuat environment python dengan mengetik ```python3 -m env myenv```
2. Aktivasi environment dengan mengetik ```myenv\Scripts\activate```
3. Clone repository ini ke environment yang sudah dibuat tadi
4. Install requirements dengan mengetik ```pip install -r requirements.txt```
5. Jalankan aplikasi dengan mengetik ```streamlit run dashboard.py```
